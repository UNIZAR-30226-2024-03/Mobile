# Frontend Móvil para la aplicación "PlayBeat", Proyecto Software 23-24 - UNIZAR

## Requirements
- Node 21
- JDK 17
- Android emulator (incluido con Android Studio)

## Instalación

```
npx react-native init PlayBeat
cd PlayBeat
npm start
```

En caso de errores utilizar
```
npx react-native doctor
```
